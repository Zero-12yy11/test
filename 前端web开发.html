1. 绝对路径

  • 定义：从文件系统的根目录（或盘符）开始，完整描述文件或目录的位置，不依赖当前工作目录。

Windows 系统：以盘符（如C:\）或网络路径（如\server\folder）开头。 示例：C:\Users\admin\Documents\file.txt

Linux/macOS 系统：以根目录/开头。 示例：/usr/local/bin/python

2. 相对路径

  • 定义：相对于当前工作目录（或指定参考位置）的路径，需结合当前位置计算实际路径。

  • 语法特征：

使用.（当前目录）、..（上级目录）或直接目录名表示层级关系。 示例：./docs/file.txt（当前目录下的docs文件夹）

例：../images/logo.png（上级目录的images文件夹）

- 基本类型： string（字符串） 、 number（数字） 、 boolean（布尔） 、 null 、 undefined 。

- 引用类型： object（对象） 、 array（数组） 、 function（函数） 。

4. Ajax**（异步通信技术）**

作用：在不刷新页面的情况下与服务器交换数据

javascript

// 创建XMLHttpRequest对象

let xhr = new XMLHttpRequest();

// 配置请求

xhr.open('GET', 'https://api.example.com/data', true);

// 处理响应

xhr.onload = function() {

if (xhr.status >= 200 && xhr.status < 300) {

// 解析JSON响应

let data = JSON.parse(xhr.responseText);

console.log(data);

// 更新DOM

document.getElementById('result').innerHTML = data.message;

} else {

console.error('请求失败');

}

};

// 错误处理

xhr.onerror = function() {

console.error('网络错误');

};

// 发送请求

xhr.send();

// 现代方式：使用Fetch API

fetch('https://api.example.com/data')

.then(response => response.json())

.then(data => {

console.log(data);

document.getElementById('result').innerHTML = data.message;

})

.catch(error => console.error('错误:', error));

```
